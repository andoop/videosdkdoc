* ##### 可以通过`IndependentVideoManager`设置全局监听（回调函数），视频的加载和播放状态，都会通过这个监听通知给开发者。

* ##### 设置方法如下：

```java
 public void addIndependentVideoListener(IndependentVideoListener listener)
```

* ##### `IndependentVideoListener`接口定义如下：

```java
public interface IndependentVideoListener {
    public void videoDidStartLoad();//进入播放界面 - 视频开始加载

    public void videoDidFinishLoad(boolean isFinished);//进入播放界面 - 视频加载完成

    public void videoDidLoadError(String error); //进入播放界面 - 视频加载失败

    public void videoDidClosed();//退出整个播放界面，返回本应用

    public void videoCompletePlay(); //进入播放界面 - 视频播放完成，或手动关闭（视为用户完成了任务，可以获取奖励）

    public void videoPlayError(String error);//进入播放界面 - 播放过程中出错

    public void videoWillPresent(); //进入播放界面 - 视频开始播放

    public void videoVailable(IndependentVideoAvailableState availableState);//通过availableState可以检查缓存状态
}
```

* ##### `IndependentVideoAvailableState`枚举类定义如下：

```java
public enum IndependentVideoAvailableState {
    /**
     * 视频正在下载
     */
    VideoStateDownloading,
    /**
     * 视频已成功缓存
     */
    VideoStateFinishedCache,
    /**
     * 无可播放的视频
     */
    VideoStateNoExist

}
```

* ##### 实例

```java
 IndependentVideoManager.newInstance().addIndependentVideoListener(new IndependentVideoListener() {
            @Override
            public void videoDidStartLoad() {
                Log.e(">>>>>", "demo videoDidStartLoad");
            }

            @Override
            public void videoDidFinishLoad(boolean isFinished) {
                Log.e(">>>>>", "demo videoDidFinishLoad"+isFinished);
            }

            @Override
            public void videoDidLoadError(String error) {
                Log.e(">>>>>", "demo videoDidLoadError");
                Toast.makeText(MainActivity.this, error, Toast.LENGTH_SHORT).show();
            }

            @Override
            public void videoDidClosed() {
                Log.e(">>>>>", "demo videoDidClosed");
            }

            @Override
            public void videoCompletePlay() {
                Log.e(">>>>>", "demo videoCompletePlay");
            }

            @Override
            public void videoPlayError(String error) {
                Log.e(">>>>>", "demo videoPlayError");
                Toast.makeText(MainActivity.this, error, Toast.LENGTH_SHORT).show();
            }

            @Override
            public void videoWillPresent() {
                Log.e(">>>>>", "demo videoWillPresent");
            }

            @Override
            public void videoVailable(IndependentVideoAvailableState availableState) {
                switch (availableState) {
                    case VideoStateDownloading:
                        Log.e(">>>>>", "demo VideoStateDownloading");
                        //视频正在缓存中
                        break;
                    case VideoStateFinishedCache:
                        Log.e(">>>>>", "demo VideoStateFinishedCache");
                        //有可播缓存视频
                        break;
                    case VideoStateNoExist:
                        Log.e(">>>>>", "demo VideoStateNoExist");
                        //无可播缓存视频
                        break;
                }

            }
        });
```
